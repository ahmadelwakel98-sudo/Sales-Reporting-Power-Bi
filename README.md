# Sales-Reporting-Power-Bi
Sales Reporting Dashboard (Power BI)
Overview
This project involves building an interactive sales dashboard using Power BI. The goal was to create a user-friendly reporting solution that enables real-time tracking of sales KPIs and supports data-driven decision-making.
Data Modeling
•	Imported and connected multiple datasets into Power BI
•	Designed data model and created relationships between tables
•	Optimized data structure for performance and usability
DAX & Calculations
•	Created calculated measures using DAX
•	Implemented KPIs such as Total Sales, MTD, Growth %, and Target Achievement
•	Developed time intelligence calculations for trend analysis
•	Handled dynamic calculations based on filters
Dashboard Design
•	Designed interactive visuals including charts, cards, and tables
•	Used slicers to allow user-driven filtering
•	Organized layout for clarity and usability
•	Focused on clean and professional UI design
Analysis & Insights
•	Tracked sales performance across time, customers, and categories
•	Identified trends and seasonal patterns
•	Detected performance gaps and underperforming areas
•	Provided insights to support business decisions
Outcome
The dashboard provides an interactive and efficient reporting tool that enhances visibility into sales data, improves analysis speed, and enables better business decision-making.
